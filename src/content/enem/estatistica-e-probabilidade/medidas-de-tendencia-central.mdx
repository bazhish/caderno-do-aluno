## Aula Teórica Completa

As **medidas de tendência central** resumem um conjunto de dados em um único valor que representa,
de alguma forma, o "centro" ou o "típico" daquele conjunto. As três principais são média, mediana e
moda — e cada uma "resume" os dados de um jeito diferente.

### Média: a soma dividida pela quantidade

A **média aritmética** é calculada somando todos os valores do conjunto e dividindo pelo número de
valores. É a medida mais conhecida, mas tem uma fraqueza importante: é muito sensível a **valores
extremos** (outliers) — um único valor muito alto ou muito baixo pode distorcer bastante a média,
fazendo com que ela não represente bem o "valor típico" do grupo.

### Mediana: o valor do meio

A **mediana** é o valor que fica exatamente no meio do conjunto, depois de todos os dados serem
**ordenados** (do menor para o maior). Se a quantidade de valores for **ímpar**, a mediana é o
valor central único. Se for **par**, a mediana é a **média dos dois valores centrais**. A grande
vantagem da mediana é que ela **não é distorcida** por valores extremos, sendo mais representativa
em conjuntos com grande dispersão ou outliers (como um grupo de salários com um diretor ganhando
muito mais que os demais).

### Moda: o valor mais frequente

A **moda** é simplesmente o valor que aparece com **maior frequência** no conjunto de dados. Um
conjunto pode ter uma moda, mais de uma moda (bimodal, trimodal) ou nenhuma moda (se todos os
valores aparecerem com a mesma frequência).

### Escolhendo a medida certa para cada situação

Não existe uma medida "melhor" em absoluto — a escolha depende do tipo de dado e do objetivo da
análise. Para dados numéricos sem outliers relevantes, a média costuma ser suficiente. Quando há
valores extremos que poderiam distorcer a interpretação (como salários muito discrepantes), a
mediana é mais indicada. Para identificar o valor mais comum (como o tamanho de roupa mais vendido
em uma loja), a moda é a medida certa.

### O que a diferença entre as três medidas revela

Quando média, mediana e moda de um conjunto são muito próximas, isso geralmente indica uma
distribuição de dados relativamente **simétrica**. Quando elas divergem bastante, é sinal de que
os dados estão **assimétricos**, possivelmente com outliers — um alerta para não confiar
cegamente em uma única medida na hora de interpretar um conjunto de dados.

**Média** = soma de todos os valores ÷ quantidade de valores — é sensível a **valores extremos**

dos dados — sinal de alerta para não confiar isoladamente na média em uma análise.
