## Aula Teórica Completa

Uma **função afim** (também chamada de função do 1º grau) é qualquer função que pode ser escrita na
forma **f(x) = a·x + b**, onde **a** e **b** são números fixos (constantes) e **x** é a variável de
entrada. É o modelo matemático ideal para representar qualquer situação que cresce (ou decresce) de
forma **constante** — ou seja, sempre na mesma quantidade a cada unidade de variação de x.

### O que representam "a" e "b"

O coeficiente **a**, chamado de **coeficiente angular** ou **taxa de variação**, indica o quanto
f(x) muda a cada unidade que x aumenta. O coeficiente **b**, chamado de **coeficiente linear** ou
**valor inicial**, é o valor de f(x) quando x = 0 — é o que você já tem (ou já paga) antes de
qualquer variação começar a contar.

### O gráfico: sempre uma reta

O gráfico de qualquer função afim é sempre uma **reta**. Se **a > 0**, a função é **crescente**
(a reta "sobe" da esquerda para a direita). Se **a < 0**, a função é **decrescente** (a reta
"desce"). O ponto onde a reta cruza o eixo vertical (eixo y) é sempre **(0, b)** — basta substituir
x = 0 na fórmula para confirmar: f(0) = a·0 + b = b.

### Como montar a função a partir de um problema

O passo mais importante para resolver questões de função afim é identificar corretamente, no
enunciado, quem é o **valor fixo/inicial** (vira o "b") e quem é a **taxa que se repete a cada
unidade** (vira o "a"). Em planos de telefonia, por exemplo, a taxa fixa mensal é o "b", e o valor
por minuto excedente é o "a".

### Crescimento e decrescimento no cotidiano

Situações de **acúmulo** (poupar dinheiro, encher um tanque) tendem a gerar funções crescentes
(a > 0). Situações de **consumo ou esvaziamento** (gastar saldo, esvaziar um reservatório) tendem
a gerar funções decrescentes (a < 0, já que o valor final diminui a cada unidade de tempo).

**Função afim: f(x) = a·x + b** — a é a taxa de variação (coeficiente angular), b é o valor

Se **a > 0**, a função é **crescente**; se **a < 0**, é **decrescente** — o sinal de "a" define a
direção da reta no gráfico.

substituir x = 0 na fórmula.

Para montar a função a partir de um problema real, identifique o **valor fixo** (vira b) e a
**taxa que se repete a cada unidade** (vira a) — esse é o passo decisivo para não errar a montagem.

Situações de **acúmulo** tendem a gerar a > 0 (crescente); situações de **consumo/esvaziamento**
tendem a gerar a < 0 (decrescente).
